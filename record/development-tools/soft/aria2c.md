# 常用快捷键
- -x n：使用 n 个线程下载文件
- -s n：在下载前等待 n 秒
- -o filename：指定下载文件的文件名
- -d directory：指定文件下载的目录
- -c：断点续传
- -m n：最大下载连接数为 n
- –max-download-limit=100K：限制下载速度为 100K/
